About Project:

Now Cold Emailing is just only one click away, no overhead of re-typing the whole email or copy-pasting previously used emails. No need to login into your different accounts just to send emails, all of it can now be managed from one account only!

  • Bootstrapped frontend & backend project setting up ts-node-dev, typescript for backend, docker containerization, AWS SDK V3, GitHub Actions for CI/CD with self-hosted runners, swagger, winston/morgan loggers, reCAPTCHA, helmet, API rate limiters, user-agent parser, postman, dev & prod environments, NgRx, setting up API Interceptors, Route Guards, handling cookies/caching mechanism, etc.
  • Integrated BullMQ and built a highly reusable architecture of queues and workers for job scheduling to send emails at a specific time, Integrated with Redis Cloud DB to save the metadata for job scheduling by BullMQ worker.
  • Created a real-time CI/CD pipeline to deploy new GitHub changes automatically on code push events by building & publishing the docker image on Docker Hub and using the GitHub-hosted & self-hosted runner on AWS ec2 to keep listening for any job to be executed (fetching new image from Docker Hub and Starting a new container).
  • Created REST API & UI for sending scheduled or real-time emails with chosen email credentials and email templates to reduce the manual effort of writing all the email content from different user email accounts.
  • Created a secure backend authentication mechanism using JWT access/refresh token to restrict API access and auto re-login using tokens.
  • Created CRUD APIs for email credentials, templates, attachments, and Authentication/Authorization features along with UI in Angular.
  • Integrated MongoDB for development & production environments, built well-structured Mongoose Schemas Design for Data Modelling.
  • Architected a well-structured express backend codebase, segregating controllers, routers, middlewares, standardizing API success/error responses and versioning API endpoints, async handlers, EJS email templates, Zod schema validation middlewares, typescript interfaces, mongoose schemas, and other utility configurations
  • Build API for IP Address details fetch, static map image with custom marker, and user device details to send real-time updates on registered routes for critical events such as attempts to reset the password (email link valid for specific time & live user IP/Geo Location /Device Details).
